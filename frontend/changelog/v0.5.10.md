# Changelog v0.5.10 - 2026-06-02

### Fixed
- **Install Endpoints wizard: step 4 now actually shows.** Clicking "Install" on step 2 now advances to step 3 (the confirmation screen). Previously the step never advanced, so the user never saw the installing/done animation.
- **Animated confirmation screen.** Step 3 now shows a spinner during install, then an animated check-circle with a satisfying bounce when complete. Shows model count, provider → tool summary, and config path. Error state shows a red dot with "Back" to retry.
- **Bigger "Done" button** with hover scale effect for clear completion affordance.
- **Error recovery.** If the install API call fails, the wizard returns to step 2 so the user can retry without closing the modal.
