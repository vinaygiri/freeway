# Changelog v0.4.2 - 2026-05-30

### Fixed
- **Stable tiny verdict column alignment** — Replaced the ambiguous-width `⚠️` emoji with the fixed-width visual marker `🟥` for `Unstable` verdicts. This prevents the first `❔` verdict indicator column from shifting in terminals that render warning-sign emoji differently.

### Changed
- **Documentation synced with verdict icons** — Updated the stability/column documentation and current release notes so the documented verdict palette matches the TUI: `🟩 Perfect`, `🟢 Normal`, `🟡 Spiky`, `🟠 Slow`, `🔴 Very Slow`, `🔥 Overloaded`, `🟥 Unstable`, `⚫ Not Active`, and `⏳ Pending`.
